## 🎨 Customization

### Update Profile

- **Photo:** Line ~109 in `index.html`
- **Name & Title:** Line ~110-115
- **Contact Info:** Line ~475-495
- **Social Links:** Line ~120-135

### Add Projects

Duplicate project card (Line ~290-330) and modify:

- Project image
- Title & description
- Tech stack tags
- Demo & GitHub links

### Change Colors

Edit Tailwind config (Line ~18-28):

```javascript
tailwind.config = {
  theme: {
    extend: {
      colors: {
        primary: "#3b82f6",
        secondary: "#8b5cf6",
      },
    },
  },
};
```
